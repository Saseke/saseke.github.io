加载数据集

通过Dataset和Dataloader进行数据的加载。

  1. Dataset：提供一种方式去获取数据以及其对应的label.
  2. Dataloader: 为后边的网络提供不同的数据形式----将Dataset的数据集进行分组，组合成n批次喂入神经网络中

小例子

import os.path

from torch.utils.data import Dataset
from PIL import Image

class MyData(Dataset):
    def __init__(self, root_dir, label_dir):
        self.root_dir = root_dir
        self.label_dir = label_dir
        self.data_dir = os.path.join(root_dir, label_dir)
        self.dataList = os.listdir(self.data_dir)

    # 加载图片数据以及对应的label
    def __getitem__(self, index):
        img_name = self.dataList[index]
        img_path = os.path.join(self.data_dir, img_name)
        img = Image.open(img_path)
        img_label = self.label_dir
        return img, img_label

    def __len__(self):
        return len(self.dataList)


root_dir = "/Users/yoke/PycharmProjects/pythonProject/dataset/val"
ant_label_dir = "ants"
bees_label_dir = "bees"
ant_data = MyData(root_dir, ant_label_dir)
bees_data = MyData(root_dir, bees_label_dir)

ToTensor

功能描述
可以将PIL以及numpy的图片转换成Tensor类型。目的：将别的数据封装成能够进行神经网络运行的基本单元
例子

from torchvision import transforms
from PIL import Image
from torch.utils.tensorboard import *
img_path = "/Users/yoke/PycharmProjects/pythonProject/dataset/val/ants/10308379_1b6c72e180.jpg"
img_pil = Image.open(img_path)
#  transforms如何使用
tensor = transforms.ToTensor
tensor_img = tensor.__call__(tensor, img_pil)
writer = SummaryWriter("logs")
writer.add_image("img_test", tensor_img, 1)
writer.close()

transforms常用方法

  1. ToTensor类
    功能描述
    将给定的图片转换成tensor类型
  2. Normalize类
    功能描述
    基于平均数和方差对图片进行标准化操作。注意标准化之后的图片是一张新的图片，原图片是保持不变的。
  3. Resize类
    功能描述
    将图片修改为指定的尺寸
  4. Compose类
    功能描述
    将多个transforms组合起来，构成一条链，方便执行

完整例子

import torchvision
# dataloader的使用
# 通过dataset加载数据集
from torch.utils.data import DataLoader
from torch.utils.tensorboard import SummaryWriter

img_dataset = torchvision.datasets.CIFAR10("./dataset", train=False, transform=torchvision.transforms.ToTensor(),
                                           download=True)

# 使用dataloader将dataset加载入神经网络中,本质就是将多个数据绑在一块，构成一个个的小集合
img_loader = DataLoader(img_dataset, batch_size=8, shuffle=True, num_workers=0)
print(img_loader[0])
# 遍历dataloader中的数据
writter = SummaryWriter("logs")
step = 0
for data in img_loader:
    imgs, tags = data
    writter.add_images("test_data_testt", imgs, step)
    step += 1
writter.close()

神经网络基础知识

在pytorch中每一个神经元就是一个module，数据会经历的过程是: input ---> module(forward) ---> output

神经网络骨架的搭建

骨架主要通过继承Module类来实现

import torch
import torch.nn as nn
# 模型定义: 继承nn.Module类，重写init,forward方法
class MyModel(nn.Module):
    def __init__(self) -> None:
        super().__init__()

    def forward(self, x):
        return x + 1
# 使用模型
model = MyModel()
# 输入一个tensor类型的数字1
input = torch.tensor(1)
ans = model(input)
print(ans)

卷积操作

本质就是对一张图片，使用一个卷积核进行卷积计算。主要使用到了:TORCH.NN.FUNCTIONAL.CONV2D

import torch
import torch.nn.functional as F

# 构建基础图片的二维数组
input_image = torch.tensor([[1, 2, 0, 3, 3], [0, 1, 2, 3, 1], [1, 2, 1, 0, 0], [5, 2, 3, 1, 1], [2, 1, 0, 1, 1]])
# 构建卷积核
kernel = torch.tensor([[1, 2, 1], [0, 1, 0], [2, 1, 0]])
# 对二维数组和卷积核进行reshape操作
input_image = torch.reshape(input_image, (1, 1, 5, 5))
kernel = torch.reshape(kernel, (1, 1, 3, 3))

# 调用conv2d,配置步长为1
result = F.conv2d(input_image, kernel, stride=1)
print(result)
# 配置步长为2
result2 = F.conv2d(input_image, kernel, stride=2)
print(result2)
