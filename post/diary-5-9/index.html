4.1 空指针与野指针

空指针：指向内存空间为0的指针

野指针：指向的内存空间是一块非法的内存空间

4.2 常量指针与指针常量

  • 常量指针：常量的指针，值不能发生改变

    int a = 10;
int const * p = &a;

  • 指针常量：指针位置不可以变，但指向的内存里面的数值可以变化

    int a = 10;
int * const p = &a;

  • const修饰指针，又修饰常量

    int a = 10;
const int *p = &a;

4.4 指针操作函数

int swap(int *a, int *b) {
    int c = *a;
    *a = *b;
    *b = c;
}

int main() {
    int a = 10, b = 11;
    swap(&a, &b);
    cout << a << "   " << b << endl;
}
