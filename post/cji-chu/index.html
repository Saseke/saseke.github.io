4.1 空指针与野指针

空指针：指向内存空间为0的指针

野指针：指向的内存空间是一块非法的内存空间

4.2 常量指针与指针常量

  • 常量指针：常量的指针，值不能发生改变

    int a = 10;
int const * p = &a;

  • 指针常量：指针位置不可以变，但指向的内存里面的数值可以变化

    int a = 10;
int * const p = &a;

  • const修饰指针，又修饰常量

    int a = 10;
const int *p = &a;

4.3 指针遍历数组

    cout << arr <<endl;

    cout << *p << endl;
    p++;
    cout << *p << endl;

4.4 指针操作函数

int swap(int *a, int *b) {
    int c = *a;
    *a = *b;
    *b = c;
}

int main() {
    int a = 10, b = 11;
    swap(&a, &b);
    cout << a << "   " << b << endl;
}

5. 结构体

5.1 结构体定义的三种方式

1. 申明
  Student s;
2. 申明+赋值
  Student s = {"yoke",1,1};
3. 申明+定义
  Struct Student{
  	string name;
  	int age;
  	int score;
}s3;

结构体变量访问属性使用.

5.2 结构体指针

结构体指针使用->对属性进行访问

Student s = {"yoke", 1, 1};
Student *p = &s;
cout << p->name << "  " << p->age << "   " << p->score << endl;

5.3 结构体指针做函数参数

struct Student {
    string name;
    int age;
    int score;
} ;

void change(Student *s) {
    s->name = "change";
}

int main() {
    Student s = {"yoke", 1, 1};
    change(&s);
    cout << s.name << endl;
}

将函数的形参改为结构体指针，可以节省开销
