  1. 结构体

5.1 结构体定义的三种方式

1. 申明
  Student s;
2. 申明+赋值
  Student s = {"yoke",1,1};
3. 申明+定义
  Struct Student{
  	string name;
  	int age;
  	int score;
}s3;

结构体变量访问属性使用.

5.2 结构体指针

结构体指针使用->对属性进行访问

Student s = {"yoke", 1, 1};
Student *p = &s;
cout << p->name << "  " << p->age << "   " << p->score << endl;

5.3 结构体指针做函数参数

struct Student {
    string name;
    int age;
    int score;
} ;

void change(Student *s) {
    s->name = "change";
}

int main() {
    Student s = {"yoke", 1, 1};
    change(&s);
    cout << s.name << endl;
}

将函数的形参改为结构体指针，可以节省开销
