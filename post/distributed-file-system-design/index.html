文件系统构成

简单的文件系统主要由三部分构成：超级块，iNode,数据块

  1. 超级块
    1. iNode的位示图
    2. dataNode的位示图
    3. INode已经使用的数目
    4. dataNode已经使用的数目
  2. INode:
    1. 所占用的所有磁盘块号
    2. 目录还是文件（文件的类型）
    3. 目录名称or文件名称
    4. 目录大小或者文件大小
    5. 权限信息
    6. 修改，更新时间
  3. DataNode:
    1. 文件数据内容
    2. 块数据的元数据: 校验和，生成戳(块的唯一标识)
    3. 有唯一的namespace ID,software版本（必须与NameNode版本一致）
    4. 盘块的唯一标识
